JSON schema reference (click to expand)
Global fields:
theme: cream | dark-minimal | dark-purple | dark-magenta
brand: text shown as vertical side label
titleSize: font size in px at 1080w (default 94)
staggerMs: ms delay between words/lines/chars (default 110)
textAnim: words | lines | chars | none (title entry mode, default words)
bodyAnim: fade | slide | none (subtitle/body/cta entry, default fade)

Per-slide overrides (all optional):
anim: slam | drift | bounce | snap | rise | fade-stagger (overrides global preset)
textAnim: words | lines | chars | none
titleSize: number
titleColor: hex color e.g. "#ff6600"
bg: hex color override for background
bgImage: URL or data:image/... base64
